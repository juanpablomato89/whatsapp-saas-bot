# WhatsApp SaaS Bot

Multi-tenant WhatsApp customer support automation platform for Latin American SMEs.

## Tech Stack

- **Backend:** .NET Core 10 · ASP.NET Core · REST API
- **Frontend:** Angular 20
- **Messaging:** WhatsApp Business API
- **Architecture:** Multi-tenant SaaS

## Features

- Automated customer support via WhatsApp
- Multi-tenant: each business gets its own isolated workspace
- Configurable automated responses and flows
- Built for Latin American SMEs (PYMEs)
